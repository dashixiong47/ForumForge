# Locales

This directory stores source translation files for built-in UI text.

- `zh-CN.json`: Simplified Chinese source strings.
- `en-US.json`: English source strings.

Runtime translations are stored in D1 tables (`languages` and `translations`) so admins can edit them from `/admin/i18n`.
These JSON files are the clean repository baseline for new installs, reviews, and future import/export tooling.
